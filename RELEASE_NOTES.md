# Release Notes - OMO Profile Manager

## Version 0.0.1

### Release Date
March 17, 2026

### What's New

#### Features
- Profile CRUD operations - Create, read, update, and delete configuration profiles
- Profile application with automatic backup - Safely switch between profiles
- Import/export functionality - Share profiles with team members
- Theme switching - Dark and light mode support
- Keyboard shortcuts - Quick access to common actions

#### Improvements
- Initial release with core functionality
- Cross-platform support (macOS, Windows, Linux)
- Electron-based desktop application
- Modern React + TypeScript architecture

### System Requirements

#### macOS
- macOS 10.12 or later
- Apple Silicon (ARM64) or Intel (x64) processor

#### Windows
- Windows 10 or later
- 64-bit processor

#### Linux
- Ubuntu 18.04+ / Fedora 30+ / Debian 10+
- 64-bit processor

### Installation

#### macOS
1. Download the `.dmg` file
2. Open the `.dmg` and drag "OMO Profile Manager" to Applications
3. Launch from Applications folder

#### Windows
1. Download the `.exe` installer
2. Run the installer and follow prompts
3. Launch from Start Menu or Desktop shortcut

#### Linux
1. Download the `.AppImage` file
2. Make executable: `chmod +x OMO-Profile-Manager-*.AppImage`
3. Run directly: `./OMO-Profile-Manager-*.AppImage`
